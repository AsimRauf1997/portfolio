Plantistan

Our project is a mobile application from where users can buy plants and seeds from the nurseries. Our application is a place from where users will be able to buy plants and seeds. The User will login as a buyer and nursery owners will be logged in as a seller. By logging in the owner will be able to add, delete and update their plants and additionally they will be able to set the prices themselves. There will be different categories from where user can select the plant and seed to buy the categories will be like plants, flowers, fruit plants, seasonal plants.

Technologies Include:

  • React Native
  • ExpressJS
  • MongoDB
  • NodeJs

E-Parking

This Project was developed as my semester project.The main purpose of E-Parking is to reduce the manual work and waiting time of the customer. The customer simply registers in the site and after registration he/she then login into the system and he/she check for the available slots, Then the customer reserves the slot by giving the details like name, address, date and time. After completing the reservation, the customer simply went to that parking stall and parks the vehicle there.

Technologies Include:

  • React Native
  • MongoDB
  • NodeJs
